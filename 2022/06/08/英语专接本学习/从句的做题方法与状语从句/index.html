英语-从句的做题方法与状语从句

发表于 2022-06-08 分类于 英语 阅读次数： Valine：
为考试做的准备

从句的做题方法

  • 定语从句中没有what和how。
  • 情态动词+动词原型=谓语动词。

练习

  • （对应第一条）形容人用of，形容事物用for。

  • in short 总而言之

  • D：全面——可以直接蒙这个（不会的话）。

  • few+可数名词 little+不可数名词！

状语从句

综合练习

  • 先行词即指人又指物只能用that。

  • 题里没有先行词，先行词必不能省，引导词作宾语可以省。
  • Be interested in 对感兴趣

写句子

  1. 找到句子中的相同部分。
  2. 用引导词替换相同部分。

  • whether…or not .. 固定搭配
  • whether与if同在，选whether。

  • 同位语从句中一般用that。

  • 这个问题是，有不确定的含义，选D。

  • 我好奇。。包含疑问色彩。

  • 包含疑问色彩。

  • 介词后面却宾语。

  • If只能引导宾语从句。

词汇

  1. Boiler n 锅炉 boil v 煮沸
  2. Freezing adj 极冷的 结冰的 Freeze v冻结 -froze-frozen v结冰
  3. Pour “泼” v倾倒
    It never rains but it pours. - 祸不单行。
  4. Telegram n 电报
